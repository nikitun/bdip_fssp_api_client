# Клиент API базы данных ФССП

Проект создавался для тестирования API ФССП v1.0. Начат и завершен в ноябре 2020.

Приложение осуществляет поиск информации о физическом лице в базе данных Федеральной службы судебных приставов по его параметрам:
- № региона;
- Имя;
- Фамилия;
- Отчество;
- Дата рождения в формате dd.mm.YYYY.

Или по его Id:
- Id контакта;
- № региона.

Результат не парсится, возвращаемый json выводится целиком.
